# MLP Forward Notes

## 核心公式

- 两层 MLP：$h=\phi(XW_1+b_1)$，$\hat y=hW_2+b_2$。
- 如果 $\phi$ 是 ReLU，则 $\phi(z)=\max(0,z)$。

## 易错点

- bias 广播维度不清楚，导致单样本和 batch 输入表现不一致。
- 把 ReLU 放在第二层之后会改变题目定义。
- 不缓存中间值，后续 backward 难写。
- 多层线性层如果不加激活，本质仍等价于一个线性层。

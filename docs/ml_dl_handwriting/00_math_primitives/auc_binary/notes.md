# Binary AUC Notes

## 核心公式

- $AUC=\int_0^1 TPR(FPR)\,dFPR$。
- 二分类排序视角：$AUC=P(s^+>s^-)+0.5P(s^+=s^-)$。

## 易错点

- 把准确率当成 AUC；AUC 衡量排序能力，不依赖某个固定阈值。
- 忘记处理相等分数，ties 通常按 0.5 计。
- 只有一个类别时 ROC-AUC 没有定义。
- AUC 高不代表概率校准好，也不代表业务阈值下 precision 一定高。

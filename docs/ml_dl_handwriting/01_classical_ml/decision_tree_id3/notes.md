# Decision Tree ID3 Notes

## 核心公式

- $H(D)=-\sum_c p_c\log p_c$。
- $Gain(D,A)=H(D)-\sum_v\frac{|D_v|}{|D|}H(D_v)$。

## 易错点

- 条件熵忘记按子集样本占比加权。
- log 底数不影响特征排序，但会影响信息增益数值。
- 连续特征需要额外找切分点，本模块只处理离散特征。
- ID3 偏好多取值特征，工程中常用增益率或正则化限制。
